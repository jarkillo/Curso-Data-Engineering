### Insertar datos JSON

### Consultar datos JSON
```python

    conn,
    tabla="eventos",
    columna_json="datos",
    campo="user_id",
    valor=123

    print(f"ID: {evento['id']}, Datos: {evento['datos']}")

```

### Actualizar campo JSON
